Description

Source catalogue

IPHAS DR2 consists of a source catalogue which provides r', i' and Hα-band photometry for 159 million unique objects, observed across the Northern Galactic Plane between 2003 and 2012.

The source catalogue is described in an accompanying paper (Farnhill et al., in preparation). In brief, the catalogue has been constructed by cross-matching the detections from the individual bands using a maximum matching distance of 1.0 arcsec. All objects which have been detected at SNR ≥ 5 in at least one band are included. We have also carried out a photometric re-calibration to bring the magnitudes to a uniform calibration, by exploiting the overlaps between exposures.

Most objects have been observed multiple times during the survey, but the DR2 source catalogue includes only the best detection of each object. A catalogue of all other detections, as well as the image pixel data, is planned for a future data release.

The "light" and "full" version

The DR2 source catalogue is made available in two flavours: "light" and "full". The light version has a size of about 5 GB (10 columns) and contains only the basic information on each source, namely:

  • equatorial coordinates (columns ra, dec);
  • r', i' and H-alpha magnitudes measured using the default 2.3 arcsec diameter aperture (columns r, i, ha, rErr, iErr, haErr);
  • a morphological classification flag (mergedClass);
  • error flags warning for e.g. source blending and saturation (errBits).

The "full" version has a size of about 50 GB (97 columns), and adds:

  • magnitudes at a second epoch in the same night, if available;
  • magnitudes measured using different aperture sizes;
  • detailed morphological parameters;
  • information on the exposure (date, seeing, run number);
  • object positions measured in the CCD reference frame (pixel x/y).

This page contains a column definitions section with more details.

Download

The IPHAS DR2 source catalogue is distributed as a series of binary FITS tables, each covering a 5x5 deg2 area. There is a light (10 columns; 5 GB) and a full version (97 columns; 50 GB). At present the catalogue cannot be queried online, but we are working to make this possible.

You may download all the files of the light catalogue (5.4 GB) to a local directory using the wget command:

wget -c -r -np -nH --cut-dirs=4 --accept=fits http://stri-cluster.herts.ac.uk/~gb/iphas-dr2/catalogue/light/

To obtain the full catalogue (50 GB), use this command:

wget -c -r -np -nH --cut-dirs=4 --accept=fits.gz http://stri-cluster.herts.ac.uk/~gb/iphas-dr2/catalogue/full/

The coordinate given is obtained from the astrometric measurement in the r'-band exposure. If the source is undetected in r', then the i' or Hα-band coordinate is given.
dec float degrees J2000 Declination. See comments above.

rErr float mag Uncertainty for r. Does not include systematic errors.

iErr float mag Uncertainty for i'. Does not include systematic errors.
ha float mag Default H-alpha magnitude using a 2.3 arcsec diameter aperture. Calibrated in the Vega system.
haErr float mag Uncertainty for ha. Does not include systematic errors.
mergedClass short Image classification flag (1=galaxy, 0=noise, -1=star, -2=probableStar, -3=probableGalaxy, -9=saturated).
errBits long Bitmask used to flag a bright neighbour (1), source blending (2), saturation (8), vignetting (64), truncation (128) and bad pixels (32768).

  • CASU dqc interface for querying image data

Non-consortium members can get a taste of IPHAS imaging data by going to the CASU dqc interface. This can be searched by specifying just a position and a search box size (other constraints can be selected but are not essential). On launching a query, a new window will open showing if there are imaging data available for the position specified. If there are, visitors to the page can reselect box size and request image cut-outs, which will be displayed in a further window. It is also possible to download the reduced images from this site.
